# チャットアプリ設計書

## 1. 要件定義書

-   GUIはGradioを使って作成する。
-   相手の返信は、こちらの返信に対して、適当に何パターンか相槌をうつ形でOK。

## 2. 設計書

### 2.1. 概要

このチャットアプリは、Gradioを使用してGUIを作成し、ユーザーがメッセージを送ると、相手が適当な相槌を返すシンプルなチャットアプリケーションです。

### 2.2. 機能

-   メッセージの送受信
-   相手からの相槌の表示

### 2.3. クラス構成

-   `ChatApp`: GradioのGUIとチャットのロジックをまとめるクラス。
    -   `__init__`: GradioのInterfaceを初期化する。
    -   `respond`: ユーザーのメッセージを受け取り、相手の相槌を生成して返す。

## 3. 実装

-   `src/main.py`に`ChatApp`クラスを実装する。
-   GradioのInterfaceを使用してGUIを作成する。
-   相手の相槌は、いくつかのパターンからランダムに選択する。

## 4. テスト

-   `tests/test_main.py`にテストコードを作成する。
-   `ChatApp`クラスの`respond`メソッドをテストする。
-   pytestを使用してテストを実行する。
